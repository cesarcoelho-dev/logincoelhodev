# Guia de Estilo
## Fontes
Este projeto utiliza uma única fonte e você pode importar diretamente do google:

Jakarta Sans:
```
@import url('https://fonts.googleapis.com/css2?family=Plus+Jakarta+Sans:wght@200;400;700&family=Ubuntu&display=swap');
```
